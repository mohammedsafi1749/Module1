## ðŸ§  Algorithm
1. Get an input from the user.

   - Else, print `"ODD"`.
4. End the program.

## ðŸ§¾ Program

if num % 2 == 0:

    print(f"{num} is an Even number")

else:

    print(f"{num} is an Odd number")
